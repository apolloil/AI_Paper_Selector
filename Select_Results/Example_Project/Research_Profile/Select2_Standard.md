# Select2 Standard

这个文档用于描述 Stage 2 的严格终筛标准。Stage 2 会在候选论文之间做横向比较，适合写得更具体、更个人化。

## 终筛目标

- 从 Stage 1 候选中选出最值得精读和复现实验的论文。
- 优先选择能直接启发下一步研究设计的论文。
- 不追求覆盖所有方向，而是保留少量高确定性候选。

## 优先选择

- 论文提出的核心机制可以清楚迁移到我的研究问题。
- 论文能改变我对问题建模、训练流程或评估方式的设计。
- 方法不是单点 trick，而是有可组合、可扩展的结构。
- 摘要中已经能看出强实验、强消融或清晰理论动机。

## 优先淘汰

- 方向有趣但需要大量额外条件才能迁移。
- 方法贡献含糊，主要依赖规模、数据或 benchmark 调参。
- 与我的当前研究计划相比，阅读收益不够直接。
- 与同组其他候选相比，novelty 或可操作性较弱。
