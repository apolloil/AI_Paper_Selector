# Select1 Standard

这个文档用于描述 Stage 1 的宽松初筛标准。建议把它写成“研究兴趣 + 可迁移机制 + 不感兴趣方向”的组合，而不是只写关键词。

## 研究目标

- 我希望寻找能够启发自己研究问题的论文，而不只限于同一个应用领域。
- 优先关注方法机制、训练策略、模型结构、评估范式或数据构造方式是否可迁移。
- 如果论文只是在熟悉任务上做工程堆叠，且没有清晰机制贡献，应降低评分。

## 高优先级特征

- 方法能迁移到我的研究问题，且迁移路径明确。
- 论文解决的问题和我的核心瓶颈相似，即使应用场景不同。
- 方法简洁、可复现，能够带来新的实验设计或理论视角。
- 论文提供了强诊断、强对比或强消融，有助于判断机制是否真实有效。

## 低优先级特征

- 仅依赖大规模算力或数据规模取得提升。
- 只针对单一 benchmark 做小幅改进。
- 摘要中看不出明确方法贡献或迁移价值。
- 与我的研究目标只有表面关键词重合。

## 打分参考

- 1-4：基本不相关，或只有非常弱的表面关联。
- 5-7：有启发，但迁移路径间接，需要进一步人工判断。
- 8-10：方法机制或问题结构高度相关，值得进入下一阶段候选。
